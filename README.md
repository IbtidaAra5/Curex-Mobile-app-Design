# Curex Mobile App

**Curex Mobile App** is a high-fidelity UI/UX design and interactive prototype for a modern healthcare platform. It connects patients with doctors, pharmacies, and emergency services while providing real-time health alerts.

## Key Features

### Figma UI/UX Prototyping
- High-fidelity interfaces for **iOS and Android**.
- Fully interactive prototypes covering the complete user journey.

### Appointment Management
- Multi-step flows for **doctor consultations** and **pharmacy orders**.
- Search/filter by specialty, view profiles, select time slots, and manage appointments.

### Emergency Services
- **SOS Alert:** One-touch emergency alert with live location.
- **Ambulance Booking:** On-demand request, location confirmation, and real-time dispatch tracking.

### Innovative UI Components
- **Interactive Body Map:** Tap on a body part to find relevant doctors or departments.
- **Dynamic Elements:** Carousels and flip cards for featured doctors and provider information.

### Geospatial Tools
- Scrollable map to **locate nearby hospitals and pharmacies** with detailed info cards.

### Public Health & Alerts
- Geo-targeted **health notifications and epidemic alerts**.
- Push notifications and in-app alert center.

### Design System
- Reusable **buttons, forms, cards, and navigation components** ensuring visual consistency.

### Interactive Prototypes
- Connected screens simulating full patient journeys.
- Key flows demonstrated in **Flow 15 (Mobile)** in Figma.
